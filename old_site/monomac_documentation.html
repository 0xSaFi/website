MonoMac API

The MonoMac API binds C and Objective-C APIs from Cocoa. The documentation is avaialble at www.go-mono.com/docs</a>.

The API design for the CIL/Objective-C binding is covered in the MonoMac API document.

Samples

We have ported some sample applications from CocoaObjective-C to MonoMac. The samples require MonoMac to be installed.

The MonoMac community has produced dozens of samples, visit the up-to-date github repository for the latest in samples.

Tutorials

The tutorials on this site will guide you through the steps of getting your sample applications running from the command line.
