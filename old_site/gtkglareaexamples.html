Rotate Buttons

This example expands on many of the other examples written so far. Some of the things that I find interesting about this example that were not previously in the others follow:

  • The OpenGL Object interface, GtkGL.IGLObject
  • The GtkGL.GLObjectBase class
  • The GtkGL.Teapot class
  • The GtkGL.ObjectRotationButton class
  • The GtkGL.GLObjectRotationController class
  • The GtkGL.Rotation class
  • The GtkGL.RotationUtil class
