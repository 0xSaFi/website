MonoTouch Roadmap

This is what we would like to see happen with MonoTouch, priorities and goals will change based on what we hear from developers.

Version 1.0

  • Core Mono.
  • Standalone MonoTouch applications.
  • MonoDevelop templates for creating MonoTouch apps.
  • Core bindings to UIKit:
  • OpenTK for OpenGLES

Version 2.0

  • Silverlight to iPhone compiler.
    • There will be Limitations on it.
  • LINQ support.
  • Debugger support.
  • Expand the API coverage to cover the rest of Apple's APIs.
  • MonoTouch.framework for reuse in existing Objective-C apps.
