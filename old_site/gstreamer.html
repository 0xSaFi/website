Gstreamer


Introduction

GStreamer# (or gstreamer-sharp) is a .NET binding for gstreamer media framework. Using this binding, a programmer can create powerful applications which can playback media files or manipulate media.

Availability

GStreamer# is available for Windows and Linux.
