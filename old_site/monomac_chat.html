Mibbit.com provides the web interface to the #monomac IRC channel hosted on this page. The #monomac channel can also be accessed through the Gimpnet IRC Network (servers: irc.gnome.org, irc.gimp.net) using any IRC client. These are some of the popular desktop IRC clients:

  • MacOS X: Colloquy
  • Linux: XChat
  • Windows: mIRC
  • Cross-Platform (using GNOME) build on Mono: Smuxi
